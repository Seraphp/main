Class IpcFactory

Description
Description | Vars (details) | Methods (details)

Static class to retrieve Ipc implementation classes

Retreive already initalized InterProcess Communication imlementation classes, depending on the given type. Class checks if the requested implementation type is exists in the filesystem. Requesting such not implemented types results throwing Exception.
